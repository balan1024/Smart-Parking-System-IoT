** Smart Parking System Using IoT**

-->Description:
An IoT-based smart parking system developed to monitor parking slot availability in real time using ESP8266 and sensors. The system uses MQTT protocol to publish slot status and display availability on a dashboard, reducing traffic congestion and improving parking efficiency.

->>Technologies Used:
- ESP8266
- Embedded C
- MQTT Protocol
- IoT Sensors
- Arduino IDE
- >> Features:
- Real-time parking slot monitoring
- Automated gate control
- MQTT-based data communication
- Live slot availability display
